This directory is the source template for a bonfire instance.

Instantiate it outside the repository. The default target path is `~/.local/share/bonfire`.

The live external bonfire instance remains outside `my-island`; this repo only owns the template tree.

## Key Files

- `docs/mission-rules.md` — defines mission usability, memory rules, member sync, and coordinator decomposition
- `memory/inheritance.md` — the single authoritative inherited-memory file

## What Bonfire Is

Bonfire is not a CLI product surface. It is a local information exchange space owned by the human.

Mission maintenance after creation stays mostly human-driven. Skills remain the execution core.

## Memory

Only `memory/inheritance.md` is the default inherited memory layer. Do not expand into multi-file memory inheritance.
