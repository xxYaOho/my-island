# AGENTS

## About it (source of ideas)

在像素风格的 RPG 中, 具现化我与 agent 的互动过程. 就像 "Stardew Valley" 这款游戏一样.

## Vision

为 human 与 agent 创造一个私人岛屿空间. 在可视化的界面中进行交流与游玩, 提供游戏式正反馈的快乐感.

## Second Identity

在 My Island 中, 你还拥有多个第二身份, 用于不同 worktree 的工作安排. 

> 注意
> Human 只能指定一个第二身份, 遇到冲突时提示 Human 当前的身份是哪个
> 未指定身份时, 默认为空
